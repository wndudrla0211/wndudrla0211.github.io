Game Ideas

Mobile Game: Dream Collector

🎮 Game Details:

Genre: Casual Adventure + Puzzle

Target Age: 12+

Graphic Style: Watercolor 2D Art

📱 Core Gameplay:

Dream Collection System

Swipe to collect floating dream fragments

Each dream piece has unique colors and emotions

Different dreams appear based on time of day (mysterious dreams at dawn)

Dream Combination Mechanics

Combine collected dreams to create new stories

Mix colors to create new dreams

Special combinations unlock rare stories

Progression Elements

Character abilities improve with more dreams collected

Unlock new regions and dream types

Acquire special dream collection tools

모바일 게임: 드림 콜렉터

🎮 게임 상세:

장르: 캐주얼 어드벤처 + 퍼즐

대상 연령층: 12세 이상

그래픽 스타일: 수채화풍 2D 아트

📱 핵심 게임플레이:

1. 꿈 수집 시스템

화면을 스와이프하여 떠다니는 꿈의 조각들을 모음

각 꿈 조각은 고유한 색상과 감정을 가짐

시간대별로 다른 종류의 꿈이 출현 (새벽에는 미스터리한 꿈)

꿈 조합 메커닉

수집한 꿈을 조합하여 새로운 이야기 생성

색상 조합으로 새로운 꿈 창조

특별한 꿈 조합 시 희귀 스토리 해금

성장 요소

꿈을 많이 모을수록 캐릭터의 능력 향상

새로운 지역과 꿈의 종류 해금

특별한 꿈 수집 도구 획득
