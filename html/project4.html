About This Game

Color Match Puzzle is a memory game where players need to match pairs of colored tiles. The game features a 4x4 grid with 8 different colors, each appearing twice. Players must find all matching pairs within the time limit to win.

컬러 매치 퍼즐은 색상이 같은 타일을 찾아 매칭하는 메모리 게임입니다. 4x4 그리드에 8가지 색상이 각각 2개씩 배치되어 있으며, 플레이어는 제한 시간 내에 모든 색상 쌍을 찾아야 합니다.
