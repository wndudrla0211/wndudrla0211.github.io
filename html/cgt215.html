About This Course

Through CGT 21500, I learned fundamental C++ programming concepts using Visual Studio 2022. Through weekly assignments, I practiced and understood core programming principles including object-oriented programming, data structures, algorithms, and game logic implementation. I completed various programming projects that demonstrate understanding of C++ fundamentals and game programming concepts as required by the course.

CGT 21500 수업을 통해 Visual Studio 2022를 사용한 C++ 프로그래밍의 기초 개념을 학습했습니다. 매주 진행된 과제들을 통해 객체 지향 프로그래밍, 자료구조, 알고리즘, 게임 로직 구현 등 핵심 프로그래밍 원리들을 연습하고 이해했습니다. 수업에서 요구하는 C++ 기초와 게임 프로그래밍 개념을 보여주는 다양한 프로그래밍 프로젝트들을 완성했습니다.

Course Overview

CGT 21500 covers C++ fundamentals, Object-Oriented Programming, Data Structures and Algorithms, Game Logic Implementation, Console Applications, and Visual Studio debugging techniques.

CGT 21500 수업에서는 C++ 기초, 객체 지향 프로그래밍, 자료구조와 알고리즘, 게임 로직 구현, 콘솔 애플리케이션 개발, Visual Studio 디버깅 기법 등을 학습했습니다.
