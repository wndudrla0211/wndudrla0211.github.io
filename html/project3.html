About This Course

Through CGT 24500, I learned the fundamental game development process using Unreal Engine 5. Through weekly assignments, I practiced and understood the following core technologies. I completed a project that implements basic game mechanics and environments as required by the course. Through this process, I was able to understand the basics of game development and learn the fundamental features of Unreal Engine.

Course Overview

CGT 24500 Simple AI, Arrays, Kit Bashing Practice, Creating User Interfaces, Data Table Import, Proc Design, Game Audio Basics in Unreal
