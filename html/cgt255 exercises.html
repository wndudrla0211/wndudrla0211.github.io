About This Game

Through CGT 25500, I learned the fundamental game development process using Unreal Engine 5. Through weekly assignments, I practiced and understood the following core technologies. I completed a project that implements basic game mechanics and environments as required by the course. Through this process, I was able to understand the basics of game development and learn the fundamental features of Unreal Engine.

CGT 25500 수업을 통해 언리얼 엔진 5를 사용한 기초적인 게임 개발 과정을 배웠습니다. 매주 진행된 과제들을 통해 핵심 기술들을 연습하고 이해했습니다. 수업에서 요구하는 기본적인 게임 메카닉과 환경을 구현하는 프로젝트를 완성했습니다. 이 과정을 통해 게임 개발의 기초를 이해하고 언리얼 엔진의 기본적인 기능들을 학습할 수 있었습니다.

Exercises Overview

CGT 25500 Simple AI, Arrays, Kit Bashing Practice, Creating User Interfaces, Data Table Import, Proc Design, Game Audio Basics in Unreal

CGT 25500 수업에서는 기초적인 인공지능 구현, 배열 시스템 활용, 키트 배싱 실습, 사용자 인터페이스 제작, 데이터 테이블 가져오기, 절차적 디자인 구현, 그리고 언리얼 엔진에서의 기본적인 게임 오디오 시스템을 학습했습니다.
